# Install Creator Releases

Public update channel for [Install-Creator-Online](https://github.com/Starf0x/Install-Creator-Online) (private repo — source code lives there).

The app itself checks `version.ini` here on startup to see if a newer build is available. This repo only ever contains:

- `version.ini` — current version number, download URL, and SHA-256 of the published exe.
- `IntuneInstallCreator.exe` — the published build referenced by `version.ini`.

Nothing else. No source code, no configuration, no internal data.
